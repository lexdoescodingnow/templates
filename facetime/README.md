# FaceTime JCink Template

Hosted CSS and JavaScript for the FaceTime social template.

Use this compact snippet in JCink doHTML posts:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/lexdoescodingnow/templates@main/facetime/facetime.css">
<script src="https://cdn.jsdelivr.net/gh/lexdoescodingnow/templates@main/facetime/facetime.js"></script>
```

Paste `template.txt` into a post and edit the visible names, image URLs, call details, and transcript text. The heavier styling and the reaction-button behavior live in the hosted files.

If jsDelivr caches an older version after an update, purge:

https://purge.jsdelivr.net/gh/lexdoescodingnow/templates@main/facetime/facetime.css
https://purge.jsdelivr.net/gh/lexdoescodingnow/templates@main/facetime/facetime.js
