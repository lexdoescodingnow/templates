DROP-IN FOR AN EXISTING BANANA TEMPLATE

1. Keep the stylesheet link directly below [dohtml].
2. Add one of the following classes to the existing title or character-name element:

banana-display
banana-display banana-display--swash
banana-display banana-display--ligatures
banana-display banana-display--signature

Example:

<h1 class="banana-title banana-display banana-display--signature">BANANA</h1>

The signature class enables both Stylistic Set 01 and the custom discretionary ligatures.
