# Petal JCink Templates

Hosted CSS for the Petal social profile and post templates.

Use this stylesheet in JCink doHTML posts:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/lexdoescodingnow/templates@main/petal/petal.css">
```

Compact ready-to-paste snippets:

- `profile-template.txt`
- `post-template.txt`

The snippets keep the editable content in the post while the heavier styling lives in `petal.css`.
